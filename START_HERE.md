### **Q: Do I need to pay for GitHub?**
**A:** No! Public repositories are free. You only pay if you want private repos or advanced features.

### **Q: Will people be able to use my code?**
**A:** Yes! MIT License allows anyone to use, modify, and distribute your code. That's the point! 💪

### **Q: Can I keep it private?**
**A:** Yes, but recruiters won't see it. Better to make it public and showcase your work!

### **Q: What if I made a mistake?**
**A:** No problem! Git tracks everything. You can:
- Revert commits
- Edit history
- Delete and restart
- Just ask for help if stuck

### **Q: How much storage do I get?**
**A:** Unlimited repositories and storage (essentially). Only limit is 100GB per file.
